bootstrap-dropdown
==================

To initialize a dropdown use:

```HTML
<li class="dropdown">
    <a class="dropdown-toggle" id="drop-1" data-toggle="dropdown" href="#">
        Dropdown title
        <span class="caret"></span>
    </a>
    <ul class="dropdown-menu" role="menu" aria-labelledby="drop-1">
        <li>Child 1</li>
    </ul>
</li>
```
